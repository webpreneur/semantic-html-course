The code element

Description

The code element represents a fragment of computer code.

Possible use cases

  • Element tag names, such my as header
  • Keywords, such as this
  • Describing function and method calls, like doSomethingAmazing()
  • Mark characters like .

Keep in mind (good to know)

By default, the code element is an inline element. To display a block of code wrap your code tag inside of a pre element.

  • For variable names, use the var tag such as myObject
  • In order to display html code as text escape "<" and "&" characters as "<" and "&" respectively.

Limitations

  • There is no formal way to indicate the language of computer code being marked up.
